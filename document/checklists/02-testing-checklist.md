# Checklist Testing Manual

## Auth
- [ ] Login admin berhasil.
- [ ] Logout berhasil.
- [ ] User non-admin tidak bisa akses halaman admin.

## Aset
- [ ] Tambah aset berhasil.
- [ ] Edit aset berhasil.
- [ ] Filter status/kategori/lokasi berjalan.
- [ ] Mode table dan grid berjalan.
- [ ] Detail drawer tampil.

## Booking
- [ ] Booking berhasil dibuat jika aset tersedia.
- [ ] Booking ditolak jika tanggal bentrok.
- [ ] Kalkulasi subtotal/diskon/pajak/deposit benar.
- [ ] Status booking berubah sesuai alur.

## Invoice
- [ ] Invoice bisa dibuat dari booking.
- [ ] Invoice tidak dobel untuk booking yang sama.
- [ ] Pembayaran sebagian dan lunas berjalan.
- [ ] PDF invoice bisa dibuat.

## Maintenance
- [ ] Work order bisa dibuat.
- [ ] Checklist bisa dicentang.
- [ ] Status maintenance mengubah status aset.

## Kalender
- [ ] Booking muncul di kalender.
- [ ] Maintenance muncul di kalender.
- [ ] Filter event berjalan.

## Responsif
- [ ] Desktop 1440px rapi.
- [ ] Tablet 768px rapi.
- [ ] Mobile 390px rapi.
