# 09 — Roadmap Development

1. Fondasi: Laravel 10, Breeze, Tailwind, MySQL, RBAC, seeder user default.
2. Aset: kategori, brand, lokasi, CRUD aset, upload foto, table/grid/detail drawer.
3. Customer: CRUD, verifikasi, detail panel, histori booking.
4. Booking: wizard, availability check, transaction lock, kalkulasi harga backend, pickup/return.
5. Kalender: FullCalendar, endpoint event, filter event.
6. Invoice: generate invoice, preview, payment partial/full, status otomatis.
7. Maintenance: work order, checklist, service log, status aset.
8. Dashboard/Laporan: KPI, chart, laporan pendapatan, utilisasi, customer.
9. Hardening: policy, activity log, testing, responsive, build production.
