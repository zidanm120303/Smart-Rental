# PROMPT UI TAILWIND

Buat ulang semua halaman Smart Rental Pro menggunakan Laravel Blade + Tailwind CSS dengan desain modern premium seperti SaaS dashboard 2026.

Aturan:

- Semua menu dan teks Bahasa Indonesia.
- Sidebar kiri, topbar, search global, notifikasi, profil user.
- Card putih rounded-2xl, border slate-200, shadow halus.
- Primary color blue/indigo.
- Status badge konsisten.
- Responsive desktop/tablet/mobile.
- Gunakan komponen Blade reusable.
- Jangan gunakan teks dummy Inggris.

Halaman wajib:

1. Dashboard
2. Manajemen Aset Table
3. Manajemen Aset Grid
4. Detail Drawer Aset
5. Booking Wizard
6. Customer
7. Kalender Operasional
8. Invoice
9. Maintenance
10. Inventory
11. Staff

13. Pengaturan
